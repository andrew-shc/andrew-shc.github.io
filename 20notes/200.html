Callout Notes

Definition (always visible)

Note

Multivariate Normal. \(X \sim \mathcal{N}(\mu, \Sigma)\) has density

\[p(x) = \frac{1}{(2\pi)^{d/2}|\Sigma|^{1/2}} \exp\!\left(-\tfrac{1}{2}(x-\mu)^\top \Sigma^{-1}(x-\mu)\right).\]

Derivation (collapsed)

TipClick to expand: sampling via Cholesky

To draw \(X \sim \mathcal{N}(\mu, \Sigma)\):

  1. Compute the Cholesky factor \(L\) such that \(\Sigma = LL^\top\).
  2. Draw \(Z \sim \mathcal{N}(0, I)\).
  3. Return \(X = \mu + LZ\).

Why it works: \(\text{Cov}(LZ) = L\,\text{Cov}(Z)\,L^\top = LIL^\top = \Sigma\). 2

2 Cholesky is \(O(d^3)\) and only needs to be computed once per covariance matrix, making repeated sampling cheap.

Common pitfall (collapsed)

WarningClick to expand: \(\Sigma\) must be positive semi-definite

Passing a non-PSD matrix to np.random.multivariate_normal raises LinAlgError. A quick fix is to add a small diagonal jitter: Σ + ε·I.

Implementation note (collapsed)

CautionClick to expand: numerical precision of the log-density

Computing \(\log p(x)\) directly can underflow for high \(d\). Prefer working in log-space throughout:

from scipy.stats import multivariate_normal
logp = multivariate_normal.logpdf(x, mean=mu, cov=Sigma)

Blockquote

The multivariate Gaussian is the maximum-entropy distribution subject to a fixed mean and covariance — it is, in a precise sense, the least informative distribution consistent with second-order statistics.

Static Code

import numpy as np
from scipy.linalg import cholesky

def sample_mvn(mu, Sigma, n, rng):
    """Draw n samples from N(mu, Sigma) via Cholesky."""
    L = cholesky(Sigma, lower=True)
    z = rng.standard_normal((len(mu), n))
    return mu[:, None] + L @ z

Long Equation

Full log-density of the multivariate normal expanded via eigendecomposition \(\Sigma = V\Lambda V^\top\):

\[\log p(x) = -\tfrac{d}{2}\log(2\pi) - \tfrac{1}{2}\log|\Sigma| - \tfrac{1}{2}(x-\mu)^\top\Sigma^{-1}(x-\mu) = -\tfrac{d}{2}\log(2\pi) - \tfrac{1}{2}\sum_{i=1}^{d}\log\lambda_i - \tfrac{1}{2}\sum_{i=1}^{d}\frac{\bigl(v_i^\top(x-\mu)\bigr)^2}{\lambda_i}\]

More sidenotes

The marginal of a joint Gaussian is also Gaussian.3 The conditional \(X_1 \mid X_2 = x_2\) is Gaussian with mean \(\mu_1 + \Sigma_{12}\Sigma_{22}^{-1}(x_2 - \mu_2)\) and covariance \(\Sigma_{11} - \Sigma_{12}\Sigma_{22}^{-1}\Sigma_{21}\).4

3 Marginalization just drops the unwanted dimensions: \(p(x_1) = \int p(x_1, x_2)\,dx_2 = \mathcal{N}(x_1;\mu_1,\Sigma_{11})\).

4 The expression \(\Sigma_{11} - \Sigma_{12}\Sigma_{22}^{-1}\Sigma_{21}\) is the Schur complement of \(\Sigma_{22}\) in \(\Sigma\). It appears in Gaussian process regression as the posterior variance.
